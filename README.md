## ⑤工夫した点・こだわった点

- サーバーサイドレンダリング実装
- 環境変数はenvファイルで管理
- 画像も扱いたかったのでCloud Storageを仕様

## ⑥難しかった点・次回トライしたいこと(又は機能)

- 状態管理の実装で複雑化したので、Redux等を取り入れたい
- DBと違ってJSON形式でデータを管理しないといけないため、設計が大変だった
- UIにこだわれなかったため、UIをリッチにしたい
- Google認証に着手したかった
